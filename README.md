# 🩺 Diabetes Prediction

## 📌 Overview
This project predicts whether a person has diabetes using Machine Learning techniques.  
It is built using Python and trained on a healthcare dataset.

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-learn

---

## 📊 Dataset
- Dataset used: Diabetes Dataset (`diabetes.csv`)
- Contains medical attributes like:
  - Glucose level
  - Blood pressure
  - BMI
  - Age

---

## 🚀 Features
- Predicts diabetes based on input data
- Data preprocessing and cleaning
- Machine Learning model training

---

## ▶️ How to Run

```bash
pip install pandas numpy scikit-learn
python diabetes.py
```
